# How to Generate the PDF Style Guide

## Quick Instructions

Follow these simple steps to create the PDF style guide for submission:

### Method 1: Using Your Web Browser (Recommended)

1. **Open the style guide file:**
   - Locate and double-click `style-guide.html` in the project folder
   - It will open in your default web browser

2. **Print to PDF:**
   - **Windows:** Press `Ctrl + P`
   - **Mac:** Press `Cmd + P`

3. **Configure print settings:**
   - Destination/Printer: Select **"Save as PDF"** or **"Microsoft Print to PDF"**
   - Layout: **Portrait**
   - Paper size: **A4** or **Letter**
   - Margins: Select **"None"** or **"Minimum"**
   - Scale: **100%** (Default)
   - Options:
     - ‚úÖ Enable **"Background graphics"** (very important!)
     - ‚úÖ Enable **"Headers and footers"** (optional)

4. **Save the PDF:**
   - Click "Save" or "Print"
   - Name the file: `Aurora-Yoga-Studio-Style-Guide.pdf`
   - Choose your preferred location
   - Click "Save"

### Method 2: Using Online PDF Converter

If your browser doesn't support "Background graphics":

1. Go to [https://www.html2pdf.com](https://www.html2pdf.com) or similar service
2. Upload `style-guide.html`
3. Click "Convert"
4. Download the PDF

### Method 3: Using Chrome DevTools (Advanced)

1. Open `style-guide.html` in **Google Chrome**
2. Press `F12` to open DevTools
3. Press `Ctrl + Shift + P` (or `Cmd + Shift + P` on Mac)
4. Type "print" and select "Capture full size screenshot"
5. Or use the regular print method with Chrome for best results

## Troubleshooting

### Colors not showing in PDF?
- Make sure "Background graphics" is enabled in print settings
- Try using Google Chrome browser instead
- Use Method 2 (online converter) as an alternative

### Fonts look different?
- This is normal - the PDF will embed the fonts
- The appearance should still be professional

### File too large?
- The PDF should be around 500KB - 2MB
- If larger, you can compress it using online tools
